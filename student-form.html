🔍 복기 · 리뷰 훈련 | 학생용

모르는 것을
끝까지 추적하는 훈련

복기 없이 푸는 문제는 실력 향상이 아니라 실수의 반복이다.
1
숙제 시작 전 — 진짜 이해 확인
✓
오늘 풀 문제의 개념을 먼저 떠올려봤나요?교재 보기 전, 머릿속에서 먼저 꺼내보기
✓
개념을 말로 설명할 수 있나요?설명 못 하면 → 개념 복습 후 숙제 시작
✓
지난 숙제에서 틀렸던 유형을 기억하고 있나요?
2
풀이 중 — 나의 사고 과정 추적
✓
막혔을 때 최소 5분 이상 혼자 고민했나요?
✓
접근 전략을 풀이지에 메모했나요?전략 없이 계산부터 시작한 문제는 복기 대상
✓
틀린 문제에 별도 표시(★ / ?)를 해두었나요?
3
복기 — 내 사고 과정 되짚기
✓
"처음에 나는 어떻게 생각했나?"를 스스로 말해봤나요?
✓
"어디서 판단이 틀어졌는가?"를 정확히 짚어냈나요?실수 / 개념 오해 / 독해 실패 중 분류
✓
틀린 이유를 내 말로 직접 썼나요?
✍️ 틀린 이유 (내 말로 — 해설 요약 금지)
4
리뷰 — 지식을 내 것으로 재구성
✓
해설 없이 다시 한 번 풀어봤나요? (백지 테스트)
✓
풀이 과정을 처음부터 끝까지 말로 설명할 수 있나요?설명 못 하면 → 아직 내 것이 아님
✓
다른 풀이 방법이 있는지 생각해봤나요?
✓
놓친 개념을 교재에서 찾아 다시 읽었나요?
5
오늘 복기 수준 — 솔직하게
📌 복기 깊이
틀린 이유만 확인
원인까지 분석
개념까지 추적
📌 리뷰 수준
해설 다시 읽기
스스로 재풀이
말로 설명 가능
✍️ 오늘 깨달음 한 줄
복기와 리뷰가 이루어지지 않으면
수업을 이어갈 수 없습니다.
